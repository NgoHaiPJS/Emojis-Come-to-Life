# 🪄 Emojis Come to Life

Turn any string of emojis into a magical, AI-generated scene!  
Just paste emojis like 🐱☕🌧️ or 🚀🌕👽 or 🏍️🍜🌆 — click Generate — and watch Pollinations.ai Flux bring them to vivid life as artwork.

## ✨ What It Does

- Paste emoji sequences into the textarea  
- The app intelligently turns them into a creative, story-like prompt  
- Generates a beautiful, high-quality image using **Pollinations.ai Flux** model  
- Shows the image + the prompt it used  
- Playful dark cosmic UI with glassmorphism, gradients, and micro-animations  

Perfect for quick fun, sharing weird emoji combos on Zalo/WhatsApp/Discord, or just seeing how wild AI can interpret simple symbols.
